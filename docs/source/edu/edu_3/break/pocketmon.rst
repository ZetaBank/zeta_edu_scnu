1. 타이머를 시작합니다.

2. 장식대 위의 포켓몬을 로봇팔의 그리퍼를 이용하여 잡습니다.

3. 포켓몬 센터에 포켓몬을 넣습니다.

4. 최대한 짧은 시간 내에 모든 포켓몬을 잡는 것이 목표입니다.
