1. 타이머를 시작합니다.

2. 장식대 위의 링을 잡습니다.

3. 잡은 링을 말뚝에 넣습니다.

4. 큰 링부터 작은 링 순서대로 넣습니다.

5. 최대한 짧은 시간 내에 모든 링을 순서대로 넣는 것이 목표입니다.
