.. thumbnail:: /_images/education/battery2.png

* 배터리 조립

      - 배터리를 배터리 고정판에 넣고 배터리 고정판과 중간판을 M3 X 6mm 2개를 이용하여 결합
      - 배터리 전원 케이블이 스피커 방향으로 빠지게 배치 후 결합

* 지지대 결합

      - M3 X 60mm 지지대 볼트 4개를 중간판에 결합
